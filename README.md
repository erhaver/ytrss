# ytrss
add youtube channels to newsboat rss feed

## requements
[sharp](https://github.com/pystardust/shup)

## configuration

set newsboat's urls file 
Default:

```
nbconfig="$HOME/.config/newsboat/urls"
``````

## usage

```
ytrss link [name]
```

Example:

```
ytrss https://www.youtube.com/@channel \"Cool Channel\"
```
